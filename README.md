# Online Store - OTHM Level 6 Python Assignment

A Flask REST API, SQLite database, and Tkinter control centre for managing
a store's product catalogue, built for the Building Python Applications unit.

## Files
- app.py - Flask REST API and database model
- gui.py - Tkinter staff control centre
- templates/index.html - customer-facing web page
- test_app.py - automated unit and integration tests
